    CONTROLS:

    • A,D -> Left and Right
    • Space -> Jump
    • W,S -> Up and Down (while flying)
    • R -> Respawn player (if you fall or get stuck)
    • B,V -> Toggle Duild Mode and Destruction Mode
    • Z -> Toggle Flying (while in build or destruction mode)
    • Ctrl+Minus -> Reset the map in destruction mode
    • Q -> Reset checkpoints (build mode)
    • Shift -> Toggle grid in build mode
    • 0-9 -> Change the brush in build mode
      • Other controls are on screen!

    SAVE/LOAD:

    • COPY MAP DATA -> Copy current map in a text format (to share it)
    • LOAD(Next to the text input) -> Load the map inside the text input field
    • CLEAR -> Clear the text input field
    • SAVE NAME -> Enter the save's name to save/load
    • SAVE MAP -> Save the current map using the entered save name
    • LOAD MAP -> Load the map with the current save name
    • DELETE -> Delete the map with the current save name
    • Click on save in the saves list -> Uses the save name that you clicked on
